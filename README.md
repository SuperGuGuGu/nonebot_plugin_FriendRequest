# 简介

基于[Nonebot2](https://github.com/nonebot/nonebot2)的好友申请处理插件，可以更加方便管理好友申请，并可选自动通过。

# 特色功能

-   可以设置是否自动通过 好友申请/群邀请
-  （对我来说）代码看起来更易懂
# 安装方式

-   ~~nb plugin install nonebot-plugin-~~（插件还没写完，别急
-  下载文件到Plugins，然后加载运行
